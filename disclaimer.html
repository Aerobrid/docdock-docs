5. PDF Editor Limitations

The PDF editor includes search, replace, and OCR functionality that has performance limitations to prevent application instability.

  • Search & Replace: Search and replace functionality is disabled for PDF files larger than 50 pages to prevent performance issues and application crashes. For larger documents, please use alternative tools or split the PDF into smaller sections.
  • OCR Limitations: OCR (Make Searchable) is disabled for PDF files larger than 50 pages on mobile devices to prevent memory exhaustion and crashes. This feature is available only on iOS and Android platforms.
  • File Size Limits: The PDF editor has file size limits of 50 MB on mobile devices and 250 MB on desktop platforms. Files exceeding these limits cannot be opened in the editor to prevent out-of-memory errors and application instability.
  • Search & Replace Accuracy: Search and replace accuracy depends entirely on the underlying PDF's font metadata and positioning stream. Even if a PDF contains searchable text, this functionality may be inaccurate or fail to match queries if the document uses custom character encodings, non-standard font programs, or lacks standard Unicode mappings (such as missing ToUnicode tables). In such cases on mobile platforms, run the "Make Searchable" (OCR) tool first to establish a clean searchable text layer.
  • Desktop Limitations (Windows): The "Make Searchable" (OCR) tool is not supported on Windows or other desktop platforms. As a result, scanned, image-only, or custom-encoded PDFs cannot be converted to searchable documents on desktop. Please perform OCR on a supported mobile device (iOS/Android) or use external tools before importing these documents.

6. File Size Limits & Processing Constraints

Different tools in DocDock have file size limits to prevent memory exhaustion, application crashes, and device instability. These limits vary by platform (mobile vs. desktop) due to available RAM.

  • File Transfer: Peer-to-peer file transfers have a maximum limit of 2 GB per file. Larger files cannot be transferred through the app.
  • File Converter: File format conversions are limited to 100 MB on mobile devices and 500 MB on desktop platforms. Files exceeding these limits cannot be converted.
  • Document Scanner: The scanner has a file size limit of 50 MB on mobile devices and 200 MB on desktop platforms, with a maximum of 100 pages per scan. Larger documents should be scanned in sections.
  • PDF Editor: As detailed in section 5, PDF editing is limited to 50 MB on mobile and 250 MB on desktop, with OCR and search/replace disabled for files over 50 pages.
  • Batch Compression: When compressing files into ZIP archives, the total uncompressed size of all files cannot exceed 2 GB.
  • Image Editor: Image editing loads the entire image into memory. Very high-resolution images may fail to load or cause performance issues depending on device RAM.
  • Transcription: Audio and video transcription uses CPU-intensive on-device processing. Large files (podcasts, movies, long recordings) may take considerable time to process and may cause device heating or resource consumption.
